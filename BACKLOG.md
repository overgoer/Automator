# Automator — Бэклог и дорожная карта

## 📋 Контекст проекта

**Automator** — ИИ-контент креатор для Telegram канала @eddytester.

### Миссия
Автоматизировать создание контента для QA-канала: анализ последних постов → генерация нового поста через ИИ → отправка на публикацию.

### Целевое состояние
- Работает автономно 24/7
- Генерирует контент в стиле канала
- Требует минимального вмешательства человека
- Надёжно как "Toyota Land Cruiser"

## 🎯 Бэклог (приоритеты)

### 🔥 Высокий приоритет

#### 1. Email уведомления (резервный канал)
**Зачем**: Если Telegram бот недоступен, дублировать на почту.

**Задачи**:
- [ ] Создать `src/notifier/email.py`
- [ ] Добавить SMTP настройки в `.env`
- [ ] Интегрировать в `main.py` (fallback после Telegram)

**Оценка**: 2 часа

---

#### 2. Режим "черновика"
**Зачем**: Не отправлять пост сразу, а сохранять как черновик для проверки.

**Задачи**:
- [ ] Добавить флаг `DRAFT_MODE` в конфиг
- [ ] Сохранять черновики в отдельную таблицу БД
- [ ] Команда `/draft` для просмотра черновиков

**Оценка**: 3 часа

---

#### 3. Валидация сгенерированного контента
**Зачем**: Проверять что ИИ не генерит мусор.

**Задачи**:
- [ ] Проверка длины поста (мин/макс)
- [ ] Проверка на запрещённые слова
- [ ] Проверка наличия эмодзи
- [ ] Авто-регенерация при провале валидации

**Оценка**: 4 часа

---

#### 4. Веб-интерфейс для управления
**Зачем**: Удобно смотреть историю, редактировать черновики, настройки.

**Задачи**:
- [ ] FastAPI бэкенд
- [ ] Простой фронтенд (HTMX или Streamlit)
- [ ] Страница истории постов
- [ ] Страница настроек
- [ ] Кнопка "запустить сейчас"

**Оценка**: 16 часов

## 🧪 Идеи для экспериментов

1. **Генерация мемов** — ИИ создаёт идею мема + описание картинки
2. **Опросы** — Авто-создание опросов на основе постов
3. **Кросс-постинг** — Дублировать в другие соцсети
4. **Голосовые** — TTS для озвучки постов
5. **Перевод** — Перевод лучших постов на английский
